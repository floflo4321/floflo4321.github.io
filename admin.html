Espace administration

Mot de passe requis pour accéder aux devis, factures et galerie.

Demandes de devis

Les clients paient uniquement par carte bleue. Pensez à le rappeler lors de l’acceptation du devis.

Factures

Paiement par carte bleue uniquement. Ce site n’envoie pas les e-mails tout seul : après chaque facture, cliquez le bouton jaune « E-mail client » (ouvre votre messagerie) ou copiez le lien Stripe.

Où est le bouton ? Onglet Factures → créez une facture → sous chaque ligne vous avez le rectangle jaune E-mail client (il reste affiché même si la facture est marquée payée).

Aperçu du bouton (même style sur chaque facture) : E-mail client

Modèles & tarifs

Les modèles que vous ajoutez ici servent à préparer le catalogue. Sur le site public (et GitHub), c’est le fichier models.json à la racine du projet qui est affiché en priorité.

Après avoir ajouté ou modifié vos modèles : cliquez sur Télécharger models.json, puis remplacez le fichier models.json dans votre dossier du site et renvoyez-le sur GitHub (ou enregistrez-le en local).

Galerie des réalisations

Ajoutez des photos depuis votre ordinateur (ou une URL) pour que les clients les voient sur la page d’accueil.

Utilisez un fichier image (JPG, PNG…) ou une URL. Les photos envoyées depuis votre ordinateur sont enregistrées localement.

Avis clients

Les avis publiés sont visibles sur la page d’accueil (section Avis). Les demandes du formulaire arrivent par e-mail et sont aussi listées ci-dessous si le navigateur a enregistré les données localement.
